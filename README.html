fluvgeo

An R Package for Performing Fluvial Geomrphology Analysis

Package Status

Description

This package contains a wide range of functions for performing fluvial geomorphic analysis. This package is designed for use with the FluvialGeomorph Python package. The FluvialGeomorph package contains an ArcGIS toolbox that can be used to extract spatial data from high-resolution terrain data. FluvialGeomorph uses functions in this package to perform a wide range of fluvial geomorphic analysis:

  • derive synthetic stream channel dimensions
  • derive stream planform dimensions
  • help choose a bankfull elevation for ungaged streams

Install

To install the fluvgeo package, install from GitHub using the devtools package:

library(devtools)
install_github(repo = "mpdougherty/fluvgeo", build_vignettes = TRUE)
